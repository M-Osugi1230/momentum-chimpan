# Swing Tendency Study v1

## 目的

この研究は、株価の完全予測や単一の最適保有日数を求めるものではありません。

モメンタム候補について、最長約3か月の範囲で、次の傾向を確認します。

- 20営業日までの強さが40・60営業日まで続くか
- 20→40日、40→60日の追加保有に意味があるか
- 初回選定、継続選定、順位改善、順位悪化、再ランクインで差があるか
- 市場の強弱、ボラティリティ、流動性、MA20からの乖離で持続性が変わるか
- +5% / +10% / +15%と、-5% / -8% / -10%のどちらに先に到達しやすいか
- 利益へ到達する前に、どの程度の逆行を経験しやすいか
- Top10の集中型とTop30・Top100の分散型で安定性が異なるか

## 評価期間

初回は、Healthy Rank v3の独立ホールドアウトと同じ2018〜2021年を使用します。

- ウォームアップ開始：2016-10-01
- 評価期間：2018-01-01〜2021-12-31
- 業種均衡・決定論的1,500銘柄
- 5取引日ごとにランキング
- 翌取引日の調整後始値でエントリー
- 5 / 10 / 20 / 40 / 60営業日
- 往復20 / 50 / 100bp

2022〜2025年との統合は、独立ホールドアウトと探索期間を分けて表示した後に行います。

## 方法

Production、Healthy v1、Healthy Rank v3を比較します。

集計は銘柄単純平均だけでなく、シグナル日ごとの等金額平均、中央値、上下5%を除いた平均、勝率、市場断面中央値からの超過を保存します。

追加保有の傾向を見るため、以下を別々に計算します。

- 0→20営業日
- 20→40営業日
- 40→60営業日

価格経路は、評価日に実際の出来高がある価格だけを使用し、エントリーの長期空白、取引日の断絶、異常な調整価格ジャンプを除外します。

## 解釈

平均収益が高いだけでは示唆としません。

複数年で平均と5%トリム平均の方向がそろい、観測数とシグナル日数が十分なセルを「記述的に一貫した傾向」とします。これは本番昇格基準ではありません。
