# Healthy Momentum v1（シャドーランキング）

## 1. 目的

現行Momentumランキングは「強さ」を捉える一方、極端な20日上昇、移動平均乖離、出来高急増を高く評価するため、すでに過熱した銘柄が上位に集まりやすい。

全期間監査では、初回から一度でも選ばれた499銘柄・1,460選定イベントを確認した。新版Top100では、初回順位51–100位、20日騰落率5–20%、20日線乖離10%以下、出来高倍率1–2倍程度が相対的に安定し、20日騰落率20%超、20日線乖離15%超、出来高10倍超は悪化した。

Healthy Momentum v1は、単純に「上昇率が大きい順」にせず、次の二段階で選ぶ。

1. 下落・構造崩れ・過熱・流動性不足・データ異常を除外する。
2. 残った銘柄を、適度な上昇、適度な乖離、適度な出来高、相対強度、直近の継続性で順位付けする。

## 2. Eligibility Gate

以下をすべて満たす銘柄だけを順位対象とする。

| 条件 | 基準 | 目的 |
|---|---:|---|
| 売買代金 | 1億円以上 | 流動性の確保 |
| 5日騰落率 | 0%超 | 短期下落中を除外 |
| 20日騰落率 | 3–20% | 弱すぎる銘柄と過熱銘柄を除外 |
| 20日移動平均 | 株価が上 | 短期構造崩れを除外 |
| 60日移動平均 | 株価が上 | 中期構造崩れを除外 |
| 20日線乖離 | 0–10% | 行き過ぎた上昇を除外 |
| 出来高倍率 | 0.8–5倍 | 参加不足と異常急増を除外 |
| 当日騰落率 | -3%超 | 当日の急落を除外 |
| 60日高値からの下落 | -10%超 | 高値崩れを除外 |
| 企業行動・異常株価 | 未解決フラグなし | 株式分割等の誤判定を除外 |

除外理由は銘柄ごとに保存する。主な状態は `RISING_HEALTHY`、`FALLING_OR_BROKEN`、`OVERHEATED`、`ILLIQUID`、`DATA_RISK`。

## 3. Healthy Quality Score

各指標は「大きいほど良い」とせず、望ましい中間値が最大になる三角形スコアを使う。これにより、20日上昇率40%や出来高10倍を追加加点しない。

| 構成要素 | 点数 | 最適付近 |
|---|---:|---:|
| 20日騰落率 | 20 | 10% |
| 5日騰落率 | 12 | 4% |
| 60日騰落率 | 8 | 15% |
| 20日線乖離 | 15 | 4% |
| 60日線乖離 | 5 | 8% |
| 出来高倍率 | 10 | 1.5倍 |
| 当日騰落率 | 8 | +1% |
| 60日高値からの下落 | 7 | -3% |
| 市場・業種相対強度 | 15 | クロスセクション上位 |
| **合計** | **100** |  |

相対強度は市場20/60日、業種20/60日の4要素を百分位化し、30%・25%・25%・20%で合成する。相対強度だけで順位を決めず、健全な絶対トレンドと併用する。

## 4. Shadow Selection Score

```text
Healthy Selection Score
= Healthy Quality Score × 75%
+ 現行Momentum Score × 25%
```

現行スコアを25%残す理由は、年初来高値、更新継続、売買代金など既存の有効情報を完全に捨てず、極端な上昇率・出来高の影響だけを弱めるため。

## 5. 既存履歴による診断

`tools/backtest_healthy_momentum.py` は、保存済みの全ランキング履歴を読み、完全に同一の再実行を除外したうえで、現行順位とHealthy順位を1・3・5レポート先の終値で比較する。

この再生結果は、設計の妥当性を確認する診断であり、本番昇格の証拠ではない。期間が短く、閾値の着想に同じ履歴を使用しているため、過学習の可能性がある。

## 6. 本番昇格条件

本番順位、Daily Action List、ペーパー取引の入力にはまだ使わない。昇格には次をすべて必要とする。

- ライブ前向き結果100件以上
- 20レポート日以上
- 5営業日平均超過の改善が0.5ポイント以上
- 現行方式に対する勝率55%以上
- 前半・後半の双方でプラス
- 1業種除外検証でもプラス
- FDR補正後の頑健性基準を満たす
- `NEXT_AVAILABLE_SESSION_ADJUSTED_OPEN` で評価
- リポジトリ所有者の明示承認
- 別PRによる本番切替レビュー

自動昇格、自動マージ、自動注文は行わない。

## 7. 運用

`Healthy Momentum Shadow` Workflowは、Daily Momentum Report成功後および手動実行時に以下を出力する。

- `latest_healthy_top100.csv`
- `latest_production_top100.csv`
- `latest_excluded.csv`
- `latest_exclusion_summary.csv`
- `historical_comparison.csv`
- `historical_per_date.csv`
- `historical_period_stability.csv`
- `report_diagnostics.csv`
- `summary.json`

Workflow権限は `contents: read` のみ。ランキング履歴、ペーパー状態、メール、サイト、本番設定を書き換えない。
